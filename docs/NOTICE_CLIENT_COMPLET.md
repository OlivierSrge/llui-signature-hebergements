## 1. SCANNER QR PARTENAIRE (OTP WhatsApp)

### Trouver le QR
Le QR code est affiché chez le partenaire : en boutique, sur le mur, sur un flyer.

### Étape 1 : Saisir téléphone
1. Scannez le QR code avec votre téléphone
2. Un formulaire s'affiche automatiquement
3. Saisissez votre numéro WhatsApp :
   - `690123456` (9 chiffres)
   - `237690123456` (avec préfixe)
   - `+237 690 123 456` (format international)
4. Cliquez [RECEVOIR MON CODE →]

### Étape 2 : Valider OTP
1. Vous recevez un message WhatsApp : *"🔐 Votre code L&Lui Stars : 123456"*
2. Saisissez le code 6 chiffres
3. Cliquez [VALIDER MON CODE →]

### Étape 3 : Utiliser
- Vous êtes redirigé vers la page de la boutique
- Votre code est valable **48 heures**
- Le code est à usage limité (5 utilisations max)

### Quota
> ⚠️ **1 code par 30 jours** par numéro de téléphone

Si vous réessayez avant 30 jours :  
*"Prochain code disponible le [DATE]"*

## 5. FAQ CLIENT

**Q : Je n'ai pas reçu mon code OTP ?**  
A : Vérifiez WhatsApp (pas SMS). Attendez 30 secondes, puis cliquez "Renvoyer le code".

**Q : Mes points expirent-ils ?**  
A : Non. Vos points restent à vie.

**Q : Je peux utiliser ma carte chez plusieurs partenaires ?**  
A : Non. Chaque carte est liée à **un seul partenaire**.

**Q : J'ai perdu mon lien de carte ?**  
A : Contactez le partenaire ou écrivez à contact@l-et-lui.com avec votre email d'achat.

**Q : Combien de temps la confirmation prend ?**  
A : 1 à 2 heures en journée (Lun-Ven 9h-18h).
