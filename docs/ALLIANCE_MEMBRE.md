## 1. Bienvenue dans Alliance Privée

Félicitations pour votre admission. Vous faites désormais partie d'un cercle très fermé — moins de 1% des candidats sont acceptés sans dossier sérieux.

Alliance Privée n'est pas une application de rencontre classique. C'est un service d'intermédiation humaine premium, où chaque connexion est calculée, chaque rencontre organisée, et chaque échange surveillé par notre Sentinelle IA pour votre sécurité.

**Ce qui vous distingue :**
- Votre profil a été lu et approuvé par un humain
- Les personnes que vous verrez ont également été validées
- Vos échanges sont protégés et confidentiels
- Les rendez-vous sont organisés dans des lieux partenaires sélectionnés

## 4. Trouver des Profils Compatibles

### Page Profils Compatibles

Tous les membres du **sexe opposé** validés y apparaissent, triés du plus compatible au moins compatible.

> Seuls les profils avec un score ≥ 40% vous sont affichés.

### Le Score de Compatibilité

Votre score avec chaque membre est calculé automatiquement sur la base de vos **5 Piliers de Vie** :

| Pilier | Points max | Comment ça marche |
|--------|-----------|-------------------|
| Vision géographique | 20 pts | Même destination = 20 pts, FLEXIBLE = 15 pts |
| Niveau d'engagement | 20 pts | Moins l'écart est grand, plus c'est haut |
| Style de vie | 20 pts | Identique = 20, ÉQUILIBRE = 15 |
| Valeurs | 20 pts | Identiques = 20, ÉQUILIBRE = 15 |
| Ambition | 20 pts | Identique = 20, MULTIPLE = 15 |

### Niveaux de compatibilité

| Niveau | Score | Couleur | Signification |
|--------|-------|---------|---------------|
| 💎 PARFAIT | 85–100% | Or | Alignement remarquable sur tous les piliers |
| ✨ EXCELLENT | 75–84% | Vert | Très grande compatibilité |
| 💫 BON | 60–74% | Bleu | Bonne base commune |
| ⭐ MOYEN | 40–59% | Orange | Compatible mais différences notables |

### Ce que vous voyez sur une fiche profil

**Visible avant le match :**
- Photo principale (floutée)
- Prénom
- Âge
- Ville / Pays
- Profession
- Score de compatibilité
- Vision géographique + niveau d'engagement

**Caché jusqu'au match :**
- Nom de famille
- Photos nettes
- Portrait de Cœur complet
- Téléphone
- Email

## 5. Marquer votre Intérêt

### Comment faire

Sur le profil d'un membre qui vous intéresse :

1. Cliquer **"Marquer mon intérêt"**
2. Attendre la confirmation (1–2 secondes)
3. Le bouton passe à **"✓ Intérêt envoyé"**

### Ce qui se passe ensuite

**Scénario A — La personne ne vous a PAS encore marqué :**
- Elle reçoit une notification anonyme : *"Quelqu'un s'intéresse à votre profil"*
- Elle peut consulter sa section "Intérêts reçus"
- Si elle marque son intérêt en retour → **Match immédiat**

**Scénario B — Elle vous avait DÉJÀ marqué (sans que vous le sachiez) :**
- 🎉 **Match immédiat !**
- Notification email + WhatsApp aux deux
- Chat sécurisé ouvert automatiquement

### Limites par tier

| Tier | Intérêts actifs max |
|------|---------------------|
| PRESTIGE | 10 simultanément |
| EXCELLENCE | 20 simultanément |
| ÉLITE | Illimités |

> Un intérêt "actif" = PENDING (ni MUTUAL ni EXPIRED).

### Accès

`Conversations → Cliquer sur le nom du membre`

### Interface du chat

```
┌────────────────────────────────────────────┐
│  [Prénom]                    🛡 Sentinelle │
├────────────────────────────────────────────┤
│                                            │
│  ┌──────────────────────────┐              │

│  │ m'a beaucoup touché.    │              │
│  └──────────────────────────┘              │
│                                            │
│           ┌──────────────────────────────┐ │
│  14:25    │ Merci pour ce message, je   │ │
│           │ suis ravi d'entendre ça.    │ │
│           └──────────────────────────────┘ │
│                                            │
├────────────────────────────────────────────┤
│  Écrivez votre message…         [Envoyer] │
└────────────────────────────────────────────┘
```

- **Messages gauche** : ce que vous recevez
- **Messages droite** : ce que vous envoyez
- **✓** : message envoyé
- **✓✓ doré** : message lu

### Sentinelle IA — votre garde-fou

Un système automatique analyse chaque message **avant envoi**. Il bloque tout contenu contenant :

| Type bloqué | Exemple |
|-------------|---------|
| Numéros de téléphone | `+237 6XX...`, `six neuf trois...` |
| Réseaux sociaux | `Instagram`, `WhatsApp`, `Snap`, `Telegram` |
| Adresses email | `xxxx@gmail.com` |
| Tentatives déguisées | `arobase`, `at`, `point` |

Si votre message est bloqué, vous verrez en temps réel l'alerte :
> *"⚠️ Sentinelle IA : numéro de téléphone détecté"*

Le message n'est pas envoyé. Reformulez sans les éléments bloqués.

### Pourquoi cette protection ?

1. **Votre sécurité** : Évite le harcèlement hors plateforme
2. **Qualité des échanges** : Encourage une vraie connexion avant le rendez-vous
3. **Intermédiation** : L&Lui organise le premier rendez-vous dans un cadre sécurisé

L'échange de coordonnées réelles se fait **uniquement** après organisation officielle d'un rendez-vous par L&Lui.

### Indicateurs de lecture

| Indicateur | Signification |
|------------|---------------|
| ✓ | Message envoyé |
| ✓✓ blanc | Message délivré |
| ✓✓ doré | Message lu |

## 8. Organiser un Rendez-vous

### Quand demander ?

Après 10–15 échanges significatifs et une vraie connexion établie. Il n'y a pas de règle stricte — vous ressentez quand le moment est venu.

### Comment faire

1. Dans le chat, contacter L&Lui directement :
   - WhatsApp : **+237 693 407 964**
   - Objet : *"Rendez-vous Alliance Privée — Match [votre prénom] / [prénom partenaire]"*

2. Indiquer :
   - 2–3 dates disponibles
   - Ville préférée (Kribi / Douala / Yaoundé)
   - Type de lieu souhaité (dîner / café / activité)

### Ce que L&Lui prend en charge

1. Contact discret des deux membres pour confirmer les disponibilités
2. Sélection d'un lieu partenaire adapté
3. Envoi d'une invitation formelle aux deux
4. Partage mutuel des coordonnées réelles (nom complet + téléphone)

### Le jour du rendez-vous

Vous recevez 24h avant :
- Nom complet et téléphone du/de la partenaire
- Adresse exacte du lieu
- Heure de rendez-vous

Après le rendez-vous, un feedback (1–5 étoiles) vous sera demandé pour améliorer le service.

## 10. Règles de Conduite

### Ce qui est attendu ✅

- Profil authentique et à jour
- Communication courtoise et respectueuse
- Répondre sous 48h aux messages si intéressé
- Honorer les rendez-vous organisés
- Feedback honnête après rendez-vous

- Faux profil ou photos d'une autre personne

## 11. FAQ Membre

**Q : Combien de profils vais-je voir ?**
Cela dépend du nombre de membres actifs du sexe opposé avec un score ≥ 40%. Actuellement entre 8 et 15 profils selon votre profil.

**Q : La personne sait-elle que je l'ai marquée ?**
Elle reçoit une notification anonyme : *"Quelqu'un s'intéresse à votre profil"*. Elle ne voit votre identité que si elle consulte *"Intérêts reçus"*.

**Q : Puis-je retirer un intérêt ?**
Oui, dans *"Mes Intérêts"* → bouton *"Retirer"*. Impossible si le match est déjà MUTUAL.

**Q : Combien de temps pour un match ?**
Variable. Certains matchs se créent en quelques heures, d'autres prennent quelques jours. Tout dépend de l'activité de vos matchs potentiels.

**Q : Les photos sont-elles vérifiées ?**
Oui, lors de l'approbation admin. Des photos manifestement fausses entraînent un refus. Restez cependant vigilant lors du rendez-vous physique.

**Q : Puis-je bloquer quelqu'un ?**
Contactez le support WhatsApp avec l'ID du match. Un admin ferme la conversation sous 24h.

**Q : Que faire en cas de harcèlement dans le chat ?**
Prenez une capture d'écran et contactez immédiatement le support WhatsApp (+237 693 407 964). Intervention sous 24h.

**Q : Mon compte expire — mes matchs restent-ils actifs ?**
Vos conversations en cours restent accessibles 30 jours après expiration. Au-delà, elles sont archivées.

**Q : Peut-on voir combien de personnes m'ont marqué sans que je sache ?**
Oui — la section *"Intérêts reçus"* vous le montre. Chaque fois que quelqu'un vous marque, vous recevez une notification.
